About ColorSmash

Using ColorSmash:

Using ColorSmash is a very simple process. Clicking the 'Change' button will change the color of the page and display the randomly generated hexidecimal value for that color in the white box below the 'Change' button.

Why ColorSmash?

ColorSmash brings a fun way and different way of finding colors. If you can't decide on what color to use or want a new one, ColorSmash can help.

Pinning Feature:

Pinning is what makes ColorSmash special. If you like a color, but can't write it down at the moment, just Pin it and the color will appear in the Pin Board section. Along with the Hexadecimal value.

Converting to RGB:

To convert the Hexadecimal value that you have pinned, simply click the square itself and it will display the RGB value. Click the square again to display the Hexadecimal.
